## Getting Started

### Clone All Projects
```bash
git clone --recursive https://github.com/oyongjoo/projects.git
```

### Clone Specific Project
```bash
git clone https://github.com/oyongjoo/nrc-rtcm-ie.git
```

### Update Submodules
```bash
git submodule update --remote --recursive
```
